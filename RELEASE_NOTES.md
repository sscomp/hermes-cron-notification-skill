# Release Notes

## v0.1.0

第一個可公開使用的版本，完成 Hermes 定時提醒技能包的基礎封裝。

### 這版包含

- `hermes-cron-notification` skill
- `install-profile.sh` 安裝器
- profile-local `scheduler` 腳本
- `hcron` wrapper 指令
- `launchd` 自動掃描與執行流程
- 舊版 `hermes-scheduler` 遷移邏輯
- 公開版 README 與 MIT License

### 目前定位

- 目標平台：macOS
- 目標環境：Hermes profiles
- 主要通知通道：Telegram
