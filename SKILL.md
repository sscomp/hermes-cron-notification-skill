# Hermes Cron Notification Package

## Description
Package a Hermes profile-local cron reminder system as an installable skill bundle. Use the installer in this repo to copy the skill, scheduler scripts, launchd runner, wrapper command, and quick commands into a Hermes profile.

## Install

```bash
/Users/sscomp/hermes-cron-notification-skill/scripts/install-profile.sh <profile-home>
```

Useful flag:

- `--no-launchctl`
